ColumnLayout Class

  고급 기능 표시
이동 :
목차
세로 또는 가로 열에 요소를 배치합니다.

열은 기본적으로 세로 방향을 사용하지만 방향 속성을 가로로 설정하면 변경할 수 있습니다.

예

<패널>
     <ColumnLayout />
     <각 Count = "10">
         <Circle Margin = "5"Color = "Blue"/>
     </ 각>
</ Panel>
